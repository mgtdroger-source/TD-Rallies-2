Google World — Setup Guide

Overview

Google World allows the Rally App to connect to Google Drive so that entry forms and control data can be shared and collected automatically.

This setup only needs to be done once. After that, Google World will be ready for future rally projects.

Before you start

You need a Google account.

  • If you already use Gmail, YouTube, or Google Drive, you already have one.
  • If not, Google will guide you through creating a free account.

Key steps

  • Sign in to Google Drive.
  • Create a Google Apps Script project.
  • Copy the Rally Code.gs script from Admin.
  • Paste the script into Apps Script.
  • Deploy it as a Web App.
  • Copy the Web App URL back into Rally Admin.
  • Save and test the connection.

Step 1 — Sign in to Google

  1. Open your web browser.
  2. Go to drive.google.com.
  3. Sign in using your Google account.

If Google Drive opens, you are ready to continue.

Step 2 — Open Google Drive

There are two simple ways to open Google Drive.

Option A — Direct link

  1. Open your web browser.
  2. Type drive.google.com and press Enter.

Option B — Google apps menu

  1. Open a Google page in your browser.
  2. Click the 9-dot apps menu near the top right.
  3. Select Drive.

Step 3 — Create an Apps Script project

  1. In Google Drive, click New.
  2. Select More.
  3. Choose Google Apps Script.

A new Apps Script project will open.

Step 4 — Prepare the Apps Script editor

When the editor opens, you will normally see:

  • a file named Code.gs in the left panel
  • sample code in the editor area

Note: Do not worry about the sample code. You will replace it in the next step.

Step 5 — Copy Rally Code.gs from Admin

  1. Return to the Rally Admin page.
  2. Open the Entries card.
  3. Click Copy Code.gs.
  4. When the code page opens, click inside it.
  5. Press Ctrl+A to select all.
  6. Press Ctrl+C to copy.

You can now close that page.

Step 6 — Replace the default script

  1. Return to the Google Apps Script tab.
  2. Click inside the editor.
  3. Press Ctrl+A to select all existing code.
  4. Press Delete.
  5. Press Ctrl+V to paste the Rally Code.gs script.

You should now see the Rally code in the editor.

Step 7 — Save the script

  1. Click the Save icon, or press Ctrl+S.
  2. Wait until the save message disappears.

Step 8 — Deploy as a Web App

  1. Click Deploy at the top right.
  2. Select New deployment.
  3. Choose Web app.

Use these settings:

  • Execute as: Me
  • Who has access: Anyone or Anyone with the link
  1. Click Deploy.
  2. When prompted, allow the required permissions.

Step 9 — Copy the Web App URL

After deployment, Google will show a Web App URL.

Copy this address.

Step 10 — Connect Rally Admin

  1. Return to the Rally Admin page.
  2. Open the Entries card.
  3. Paste the Web App URL into the URL field.
  4. Enter your mobile number, including international prefix.
  5. Click Save & Test.

If successful, the status message will show Connected.

Google World is now ready

You do not need to work directly in Google Drive again for normal use.

After the first upload of Admin data, a folder called Time and Distance Rallies will be created in your Google Drive.

Using Google World

Entries

  • Upload Entries creates the online entry form, including Speed Group options.
  • Download Entries imports submitted entry data into the Admin page.

Controls

  • From the Controls page you can upload STC and Marshal forms.
  • You can then download responses from those forms.
  • The returned data populates the Controls page ready for review and printing.
