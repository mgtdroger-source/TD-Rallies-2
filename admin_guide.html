Admin — Entries & Rally Setup

Overview

The Admin page is the starting point for every rally. It is used to create the rally record, define rally days, speed groups, and enter competitor details. Admin is the single source of truth for rally structure and entrant data. All other pages (Schedule and Controls) read from the data created here.

Quick steps

  • Name the rally, this sets up page links and local memory recording your work.
  • Set up the rally workspace folder ready for saving a backup of your work.
  • Select your rally days, dates, starting times and leaving increment.
  • Select the number of speed groups (SG) and speeds for each group.
  • Set up your Google Drive.
  • Upload the entry data and then share the form link.
  • Entries can be fully automated or manually entered.

The page has a Normal/Compact button. Toggling this changes the page size.

Admin layout

The page has three main sections: days, speed groups and routes; the control panel; and the entries table.

Days, speed groups and Routes

  • Select the number of Rally days using the day selector, up to 7 days.
  • Dates, click the calendar icon, select the starting date, following day dates cascade, for a rest day increment the date after the rest day, further days will cascade again.
  • Starting times, click the clock icon, select the start time for each day.
  • Increment, the start time interval which can be incremented in 30 sec steps, the default is 1 minute.
  • Select the number of Speed Groups using the selector, up to 4 speed groups.
  • Adjust the speed for each Speed Group as required, the speed of a slower group cannot exceed that of a faster group, as the speed of a faster group is decremented it will decrement the speed of the slower group below.
  • The speed selected is the maximum speed that will be available when setting speeds in the route schedule, the speed allocated to an entrant and for result calculations. If a selected speed is changed allocations update accordingly.

    • Note: if a speed is increased then a review of the schedule speeds will be required, these do not automatically increment, they only decrement automatically.

  • Routes, when 2 is selected, each rally day requires two schedules. Each route must have the same number of controls such as STCs and Marshals.

With 2 routes selected the table of the Admin page gains an additional column to assign entrants to a route.

Route allocations are not subject to sort, so they must be reviewed once an entry table has been sorted by speed group. The first car away should be assigned to route 1.

Note: If 2 routes are only required for one day of the rally, then routes can be set to one for the other days.

Control panel

  • Enter Rally Title, your Rally Name, which is the first detail to be entered when commencing a new rally, next to the title is the rXXX number, it increments each time you save the project.
  • ADD ROW adds an additional entrant row to the bottom of the table. Downloaded entries automatically add rows.
  • Sort used to arrange the table of entrants into speed group order. Sort the entrant table is sorted.
  • Save save your work to a selected workspace folder.
  • Load select a Rally file from your workspace folder, note: loading will overwrite your current work, so consider saving prior to loading.
  • Print print entrant details and their day starting times.
  • Entries when selected it presents a card which is the gateway to automation, refer to Google integration below.

Toolsmunu of items.

  • Setting up your workspace, for example a folder in My documents.

    • In order to save your projects on your computer, a workspace folder is required. Selecting Set Rally Workspace opens a Windows picker, enabling you to navigate to where you want to save projects, for example a new folder in My Documents named "Rallies". Based on your chosen Rally title, a new sub folder is created within the Rallies folder when you save. Inside that folder will be your project file with its rXXX reference. Each time you save the same project, rXXX is incremented by 1 and shown on each of the program pages.

  • Save As, which enables you to save your current project giving it a new Rally Title.
  • Load, select a previously saved project.
  • New Rally, resets all the pages clearing any prior work, a new Title is required.

The Table

Entry and management of competitor details, either manually or fully automated. Entrants can be sorted by speed group, including start delays between groups. Add row button: Adds a new entrant row to the bottom of the table.

Note: Rally numbers and start times cannot be edited in the table. Rally number sequence is fixed.

Entry column and gear icon.

As each entry is added to the table an index (row number) is assigned to that entrant, later when the table is sorted the index will remain with that entrant.

Entry Row gear menu:

  • Insert a new row above or below, index numbers update maintaining sequence.
  • Copy row, entrant details excluding Rally number and times which are fixed.
  • Paste, Pasting is only allowed into an empty row.
  • Delete, when selected a confirmation is presented before the row is removed.

Note: when rows are inserted or deleted sequence numbers are updated.

Speed Group column

Clicking a Speed group cell in the table will open a selector menu from which to pick a speed group, once selected the cell will be colour coded for ease of reference. If the group speed is amended table cells will update automatically. If an entrant allocated in a slower group and that group option is removed, then that entrant will be promoted to the next higher speed group.

Note: If the omitted speed group is reinstated during the same work session, then entrants will be returned to that group.

Sort

The sort function is used to arrange the table of entries into speed groups, typically slowest to fastest. Sort should only be used when all entries have been received. Once a table is sorted, options such as Add Row, the gear icon options and download of data are no longer available. If late entries are to be accommodated, then select sort and restore the tables original order. When a table is sorted the original entry index number will remain with the entrant.

Notes

  • When sorting Rally numbers and start times remain fixed.
  • Route allocation remains with the entrant and can be updated as required.
  • Special case, when a table is sorted, an entrant's SG can be changed, but his rally number and start time will remain the same.

Google integration

When Entries is selected, a card is presented. This is the interface for setting up and interacting with Google World. Once set up, the facility is fully automated and little or no interaction with Google is required. All forms, tables and rally-based folders are fully automated.

Setup process

  • In the setup section of the card enter your mobile number which must include an international prefix for example (+27) followed by your number excluding the leading 0.
  • URL will be copied once you have completed the initial setup of Google World.
  • For detailed instructions select the Setting up guide button.
  • Once both mobile and URL have been entered select Save & test.
  • At the top of the card the detail should now change to “Connected” all the hard work is done.

Using the Entries card

When the Entries card is open there are rows of buttons, first in the lower section of the card is Setup, if not already set up click the Setting up guide button and follow the instructions.

Entry link

Until entry data has been uploaded these buttons will be greyed, but after upload they will be lit. The buttons are used to select the method of sending a link to the Rally entry form, if WhatsApp (WA) is selected you will need to have the WA app installed on your machine. When WA is selected a message and link will be sent to yourself which you can then distribute further.

WA is by far the most convenient method of sharing form links, and most people today are comfortable using WA, once all the entrant details have been gathered consider setting up a WA group which will make the distribution of the result (STC) form so much simpler.

Uploading entry data

Before uploading entry data ensure you have set up the rally Speed Groups options and speeds, to the left of upload is displayed the Rally Title, this is the text that will appear at the top of the Google entry form, the text can be edited as required.

Note: uploading of data takes a little while, so be patient a lot is being created.

Entry data

View response forms button, opens a new tab presenting the Google form responses.

Open entry data button, opens a new tab presenting the worksheet of processed Google data that is ready for download.

Note: An entrant can submit more than one response, but only their last will appear in the worksheet ready for download.

Download button, this button will be greyed until there is data ready for download, when data is downloaded it overwrites any prior data in the entries table.

Tip: review entry data worksheet before downloading to gain an appreciation of the entries received.

Application RESET

Full Reset is rarely needed. If things go wrong for any reason, follow these steps. Select F12 on your keyboard, select "Applications", in the left panel select "file://" from below "Local storage", right click file then select "clear".

The local memory is now cleared. Go to your Google Drive, and delete the folder "Time & Distance Rallies", also delete the folder from the trash bin. Now repeat the Google setting up process. Potentially you may have a corrupted saved rally file, so consider deleting that folder so you will be starting over from a totally clean base.
